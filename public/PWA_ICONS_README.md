# PWA Icons

## Required Icons

The following icon files are placeholders and need to be replaced with actual images:

- `pwa-64x64.png` - 64x64px icon
- `pwa-192x192.png` - 192x192px icon for Android
- `pwa-512x512.png` - 512x512px icon for Android (any purpose)
- `maskable-icon-512x512.png` - 512x512px maskable icon for Android
- `apple-touch-icon.png` - 180x180px icon for iOS
- `favicon.ico` - Favicon for browsers

## How to Generate Icons

You can use tools like:

- <https://realfavicongenerator.net/>
- <https://www.pwabuilder.com/imageGenerator>
- <https://maskable.app/> (for maskable icons)

## Icon Guidelines

1. **pwa-64x64.png, pwa-192x192.png, pwa-512x512.png**: Use your app logo with padding
2. **maskable-icon-512x512.png**: Must fill the entire safe zone (see <https://maskable.app/>)
3. **apple-touch-icon.png**: iOS home screen icon (180x180px recommended)
4. **favicon.ico**: Traditional browser favicon (32x32px)

All icons should use your app's branding colors matching the theme (#0ea5e9).
